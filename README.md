# My Python to Machine Learning Learning Journey

This folder documents my step-by-step journey learning Python from the very basics all the way to building my first machine learning model.

## Overview

Starting with "Hello, World!" and progressing through Python fundamentals, data science tools (NumPy and Pandas), and finally machine learning concepts, this collection represents my learning path. Each file focuses on one concept and includes comments that reflect my thinking process as I learned.

## How to Use

1. Start with `01_hello_world.py` and work through the files in numerical order
2. Each file is self-contained and can be run independently
3. Read the comments - they explain what I was learning and thinking
4. Try modifying the code to experiment and reinforce learning

## Requirements

For the basic Python files (01-12), no external libraries are needed - just Python 3.

For data science and ML files (13-25), you'll need:
- NumPy (`pip install numpy`)
- Pandas (`pip install pandas`)
- scikit-learn (`pip install scikit-learn`)
